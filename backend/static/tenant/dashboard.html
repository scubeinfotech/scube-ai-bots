WhatsApp Setup Guide

                                    
                                


                                

                                    
                                    

                                        
How it works

                                        
When a customer sends a WhatsApp message to your number, Meta/MSG91 forwards it to our webhook URL. Our chatbot processes it through the LLM and sends a reply back via the provider's API.

                                    


                                    
                                    

                                        
🔄 What You Need (Summary)

                                        

                                            

                                                
📥 FROM Provider (MSG91/Meta)

                                                
    
                                                    
  • Auth Key - for sending replies
    • 
                                                    
  • Integrated/Phone Number - your WhatsApp number
    • 
                                                

                                                
Enter these in the form below ↓

                                            

                                            

                                                
📤 TO Provider (MSG91/Meta)

                                                
    
                                                    
  • Webhook URL - for receiving messages
    • 
                                                    
  • Verify Token - for security
    • 
                                                

                                                
Copy these to your provider dashboard ↓

                                            

                                        

                                    


                                    
                                    

                                        
Step 1: Choose your WhatsApp Provider

                                        
Decide which provider you want to use:

                                        

                                            

                                                
Meta (Cloud API)

                                                
Direct integration with Meta's WhatsApp Cloud API. Best for businesses with a Meta Developer account.

                                            

                                            

                                                
MSG91

                                                
Integration via MSG91 as intermediary. Best for businesses already using MSG91 for SMS/WhatsApp.

                                            

                                        

                                    


                                    
                                    

                                        
Step 2A: Prepare Meta Credentials

                                        
    
                                            
  1. Go to Meta for Developers
    2. 
                                            
  2. Create or open your App → WhatsApp product
    3. 
                                            
  3. From the dashboard, copy your Phone Number ID and Business Account ID
    4. 
                                            
  4. Go to System User page, create a token with whatsapp_business_messaging and whatsapp_business_management permissions
    5. 
                                            
  5. Copy the generated Access Token (starts with EAA...)
    6. 
                                        

                                    


                                    
                                    

                                        
Step 2B: Prepare MSG91 Credentials

                                        
    
                                            
  1. Go to MSG91 Dashboard → WhatsApp section
    2. 
                                            
  2. Copy your Auth Key from Settings → API Keys
    3. 
                                            
  3. Copy your Integrated Number from your WhatsApp sender profile
    4. 
                                            
  4. Leave the API Endpoint URL as default unless MSG91 provides a custom one
    5. 
                                        

                                    


                                    
                                    

                                        
Step 3: Enter Credentials in Our Dashboard

                                        
    
                                            
  1. Select your provider (Meta or MSG91) using the toggle buttons
    2. 
                                            
  2. Paste the credentials into the corresponding input fields
    3. 
                                            
  3. Leave API Endpoint URL blank to use the default
    4. 
                                        

                                    


                                    
                                    

                                        
Step 4: Configure Webhook in Provider Dashboard

                                        
Copy the following from the Webhook Configuration box in our dashboard:


                                        

                                            
Callback / Webhook URL:

                                            
                                        

                                        

                                            
Verify Token:

                                            scube_wa_verify_2024
                                        


                                        
Paste these into the Webhook settings of your Meta App or MSG91 dashboard. Meta/MSG91 will send a verification GET request — our server verifies the token and responds with the challenge.

                                    


                                    
                                    

                                        
Step 5: Configure Webhook in Your Provider Dashboard

                                        
Copy these exact values to your MSG91 or Meta dashboard:

                                        
                                        

                                            
                                            

                                                
                                                

                                                    
                                                    
                                                

                                            

                                            
                                            
                                            

                                                
                                                

                                                    
                                                    
                                                

                                            

                                            
                                            
                                            

                                                

                                                    For MSG91: Go to Dashboard → WhatsApp → Configuration → paste these values → Click "Verify & Save"
                                                

                                                

                                                    For Meta: Go to developers.facebook.com → Your App → WhatsApp → Configuration → Webhooks → paste these values → Click "Verify & Save"
                                                

                                                

                                                    Select a provider in Step 1 to see specific instructions
                                                

                                            

                                        

                                    


                                    
                                    

                                        
Step 6: Click Connect

                                        
After configuring the webhook in your provider dashboard, click the Connect WhatsApp button below. Our system will verify the connection and activate your WhatsApp chatbot.

Troubleshooting

                                        
    
                                            
  • Webhook verification fails: Make sure the Verify Token in Meta/MSG91 matches scube_wa_verify_2024 exactly
    • 
                                            
  • Messages not being received: Check that your webhook URL is set correctly in the provider dashboard and the webhook is subscribed to messages events
    • 
                                            
  • Messages not sending: Verify your Access Token (Meta) or Auth Key (MSG91) is still valid and hasn't expired
    • 
                                            
  • Need to update credentials: Click Edit Configuration on the WhatsApp card to update any field
    • 
                                            
  • MSG91 API changes: If MSG91 provides a different API endpoint URL, update it in the API Endpoint URL field
    •
