# Security and Compliance Plan

## Core Controls
- Tenant isolation by design at API and data layers
- Least-privilege access for admins and services
- Encrypted data in transit and at rest
- Secret management with key rotation

## Data Governance
- Tenant-scoped retention policy
- Deletion workflow for conversations
- PII masking for logs and analytics

## Insurance-Specific Controls
- Restricted response policy for legal/coverage claims
- Mandatory disclaimer templates
- Escalation to human agent for sensitive intents

## Audit and Monitoring
- Audit logs for admin actions
- Access anomaly alerts
- Quarterly security review and penetration checks

## Incident Response
- Severity classification model
- Escalation contacts and SLA
- Post-incident RCA and remediation tracking
