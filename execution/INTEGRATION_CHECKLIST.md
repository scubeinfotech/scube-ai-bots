# Integration Checklist

## Pre-Integration
- Tenant ID created
- API key issued and stored securely
- Domain allowlist configured
- Prompt and policy approved

## Website Integration
- Widget SDK script added
- Theme config applied
- Basic user journeys verified
- Fallback and error handling verified

## Validation
- Intent coverage tests pass
- Latency and reliability checks pass
- No cross-tenant answer leakage
- Compliance checks pass (insurance tenants)

## Go-Live
- Production key enabled
- Monitoring alerts enabled
- Rollback plan documented
- Stakeholder sign-off completed
