QUICK START
================================================================================

1. Verify Installation:
   source .venv/bin/activate
   cd backend
   python -c "from app.api import whatsapp; print('✓ Ready')"

2. Configure WhatsApp:
   curl -X POST http://localhost:8000/api/whatsapp/configure/rapas \
     -H "Content-Type: application/json" \
     -d '{...configuration...}'

3. Set Webhook in WhatsApp Platform:
   URL: https://yourdomain.com/api/whatsapp/webhook/{tenant_id}
   Token: my_verify_token

4. Test with Message:
   Send a WhatsApp message to your business number

5. Check Health:
   curl http://localhost:8000/api/whatsapp/health/rapas

For details: See WHATSAPP_QUICK_START.md
